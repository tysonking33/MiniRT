# MiniRT

learn opengl from: https://learnopengl.com/Getting-started/OpenGL


Sphere:
  Center: (x, y, z)
  Radius: r

Plane:
  Normal: (nx, ny, nz)
  PointOnPlane: (x, y, z)

Triangle:
  Vertex1: (x1, y1, z1)
  Vertex2: (x2, y2, z2)
  Vertex3: (x3, y3, z3)

Cylinder:
  Axis:
    Point1: (x1, y1, z1)
    Point2: (x2, y2, z2)
  Radius: r

Cube:
  Center: (x, y, z)
  SideLength: s
